## 1. Install Tuner

Simucube Tuner is a configuration software for Simucube 3 Wheelbases and other Simucube devices. To get started [download Tuner](../../Tuner/index.md) and launch the installer.

## 2. Check cabling

If you haven't done connections yet, see [cabling](../Cabling.md).


## 3. Power on

After the installation steps are completed it is time to turn on your wheelbase. This is done by pressing the white button on the Control Box.

## 4. Software configuration

### Firmware update

When the Simucube Tuner software is started for the first time, a firmware update of the wheelbase and Link Hub is required. This is also indicated by a pop-up on the top of the screen.
Press the indication text "Update device" or "firmware update" -button in the bottom left corner to open a pop-up which indicates all the devices that can be updated. Updating the device firmware is done by pressing the update button. Close pop-up when ready.

### First startup

When the firmware is updated, Simucube 3 is seen as a new device in device panel on the left side of the application and on the "overview" page of Tuner.

For more information about getting Simucube 3 set up for your first test drive, see [Basic Setup](Setup.md).

## 5. Test Torque off button

Press the torque off button (E-stop) located on Control box. In the bottom left corner of Tuner software, torque off status should change to "pressed". To release Torque off button, turn it clockwise.

## 6. Test drive

Launch your simulator and take your first test drive. Do not use high torque mode on first test drive, to verify the rigidity of the wheelbase.
