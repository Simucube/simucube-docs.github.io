Simucube Control box allows the user to control various functions of Simucube 3 wheelbase and other Simucube devices remotely.
This chapter addresses Control box functions and configuration. For information about Control Box mounting, see [Mounting](Mounting.md)

Before making configuration changes, make sure your Control Box is connected to your Simucube 3 wheelbase. Control Box is also required to be connected for Simucube 3 to be switched on. For detailed cabling instructions, see [Cabling](Cabling.md)


## Button functions
![](assets/ControlBoxButtons.svg)

### 1: Torque off

Used as an emergency stop when all connected devices need to be quickly powered off. Press to activate, rotate clockwise to release.

### 2: Torque adjustment / Quick zero

The rotary encoder (number 2) can be used to adjust feedback strenght and other parameters of Simucube 3 wheelbases while driving. 

- Pressing / rotating the encoder knob opens an in-game adjustment window.
- Rotate the knob to adjust the selected parameter (feedback strenght as default).
- Press down (hold) and rotate the knob to change the selected parameter.
- The window will automatically close after 5 seconds of inactivity.
- *Note: If you can not see the in-game window, try adjusting your video settings from "fullscreen" to "windowed borderless" or vice versa.*

You can modify the list of adjustable parameters displayed in the in-game window by using Tuner.

### 3: Power Switch

Used to turn your Simucube 3 wheelbase on or off.

![](assets/ControlBoxPowerLight.svg#gh-light-mode-only){width=350}
![](assets/ControlBoxPowerDark.svg#gh-dark-mode-only){width=350}

### 4: Sleep / Activate

This button can be used to wake up all your Simucube devices from idle state, including pedals.

### Indicator light

On the Sleep / Activate button (number 4), an indicator LED can be found. The colours of the LED indicate following device states:

| LED Colour				| State   						|
| ------------------------  | ----- 						|
| Continous green   		| Normal operating mode 		|
| Continous purple    	    | Disabled						|
| Cyan blinking   		    | Safe mode active				|
| Red blinking	 			| Device fault  				|
| Red / Orange blinking  	| Torque off pressed			|
