## Required items to make use of Simucube 3 Wheelbases

To get started with your Simucube 3 Wheelbase you need:

* **Simucube 3 Wheelbase**
* **Simucube Link Hub (sold separately)**
* **Wheelside [Link Quick Release](../Simucube3/Quick release.md) (sold separately)**
* **A fully functioning sim racing system**
	* A sturdy sim racing rig
	* Gaming computer with an available USB port
	* [Simucube Steering wheel](https://simucube.com/steering-wheels/), or any other combatible wheel
	* Set of [Simucube pedals](https://simucube.com/pedals/), or any compatible other pedal set

## For systems with Simucube pedals

When using Simucube 3 in combination with ActivePedals you need:

* All Simucube 3 related items listed above
* One or more ActivePedal Pro or ActivePedal Ultimate
* [Teltonika Ethernet Switch](https://simucube.com/teltonika-ethernet-switch/) or another ethernet switch with enough ports for Link Hub, Simucube 3, and ActivePedals


## Optional items

* Ethernet switch *(For connecting multiple devices to Link Hub)*
* Accessories
