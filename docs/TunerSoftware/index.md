# Simucube Tuner 3.0

Tuner 3.0 is the next generation configuration software for Simucube Link based devices, including Simucube 3 and 2 wheelbases, Active Pedals, Simucube Throttle and wireless steering wheels.

![](assets/tuner3_overview.png)

## Download

[:material-download-circle: Simucube Tuner 3.0](https://downloads.simucube.com/SimucubeTunerSetup.exe) (latest release)

!!! note "Tuner 3.0 vs Tuner 2"
    Tuner 3.0 introduces a redesigned user interface and new features. If you're using older software the user interface works slightly different ways. [See the Tuner Legacy documentation](../TunerSoftware/tunerlegacy/index.md)

## What's New in 3.0

- **Simucube 3 Support**: Full configuration for Simucube 3 wheelbases
- **Modernized UI**: Redesigned interface for better usability
- **Enhanced Profiles**: Improved profile management and switching
- **Advanced Telemetry**: Better game integration and effect control
